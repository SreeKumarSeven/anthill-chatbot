Anthill IQ Chatbot

This is the deployment page for the Anthill IQ Chatbot. To integrate this chatbot with your Wix website, follow the instructions below:

Integration Instructions

Method 1: Using Wix Custom Element (Recommended)

Add the following code to your Wix site using a custom HTML element:

<div id="anthill-chatbot-container"></div>
<script>
    (function() {
        var script = document.createElement('script');
        script.type = 'text/javascript';
        script.async = true;
        script.src = 'https://your-vercel-deployment-url.vercel.app/anthill-widget.js';
        
        var entry = document.getElementsByTagName('script')[0];
        entry.parentNode.insertBefore(script, entry);
    })();
</script>

Method 2: Using an iFrame

Alternatively, you can embed the chatbot using an iframe:

<iframe 
    src="https://your-vercel-deployment-url.vercel.app/iframe.html" 
    width="400" 
    height="600" 
    frameborder="0"
    style="position: fixed; bottom: 20px; right: 20px; border: none; z-index: 9999;">
</iframe>
