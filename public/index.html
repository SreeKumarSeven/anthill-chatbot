Anthill IQ Chatbot API

Status: Online

This is the API endpoint for the Anthill IQ chatbot. The API is running and ready to accept requests.

To interact with the chatbot, use the widget in the bottom right corner of this page, or follow these integration instructions for your own website:

Integration Code for Wix:

<div id="anthill-chatbot-container"></div>
<script>
    (function() {
        var script = document.createElement('script');
        script.type = 'text/javascript';
        script.async = true;
        script.src = 'https://anthill-chatbot.vercel.app/anthill_chatbot_widget.js';
        
        var entry = document.getElementsByTagName('script')[0];
        entry.parentNode.insertBefore(script, entry);
    })();
</script>
