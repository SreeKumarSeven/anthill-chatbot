Anthill IQ Chatbot Widget

A customizable chatbot widget for Anthill IQ's Wix website with backend API integration.

Wix Integration Instructions

Add to your Wix site by embedding this code in an HTML Code element:

<script src="https://SreeKumarSeven.github.io/anthill-chatbot/anthill_chatbot_widget.js"></script>
<link rel="stylesheet" href="https://SreeKumarSeven.github.io/anthill-chatbot/anthill_chatbot_widget.css">
Important: When adding HTML code to Wix, make sure to click on "Settings" for the HTML element and turn OFF "Show on mobile" if you're encountering any issues with the chatbot on mobile devices.

Features

  • Backend Integration: Connects to Anthill IQ's backend API to provide intelligent responses
  • User Registration: Collects name and phone number before starting the chat
  • Responsive Design: Works on desktop and mobile devices
  • Modern UI: Clean, professional design that matches Anthill IQ's branding
